### 𝙃𝙤𝙬 𝙩𝙤 𝙪𝙨𝙚

This project requires [Python 2.7](https://www.python.org/download/releases/2.7/)

Requires `python-gi`

```
sudo apt-get install --reinstall python-gi
```

To run `Battery_Manager`, run: 

```
python battery.py
```
